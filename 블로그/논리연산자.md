## 논리연산자

a = True, b = False이라고 가정하겠습니다.

| Operator |              Description              |       Example       |
| :------: | :-----------------------------------: | :-----------------: |
|   and    |   논리 AND 연산. 둘 다 참일때만 참    |  (a and b) = False  |
|    or    | 논리 OR 연산.둘 중 하나만 참이여도 참 |   (a or b) = True   |
|   not    |    논리 NOT 연산. 논리 상태를 반전    | not(a and b) = True |
