### 산술 연산자

a = 10, b = 20, c = 3 이라고 가정하겠습니다.

| Operator | Description |    Example    |
| :------: | :---------: | :-----------: |
|    +     |   더하기    |  a + b = 30   |
|    -     |    빼기     |  a - b = -10  |
|    *     |   곱하기    |  a * b = 200  |
|    /     |   나누기    |  b / a = 2.0  |
|    %     |   나머지    |   b % a = 0   |
|    **    |    제곱     | a ** c = 1000 |
|    //    |     몫      |  a // c = 3   |
