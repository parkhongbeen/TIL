## 맴버연산자

a = 10, b = 10, list = [1, 2, 3, 4, 5]라고 가정하겠습니다.

| Operator |           Description            |        Example         |
| :------: | :------------------------------: | :--------------------: |
|    in    |   list내에 포함되어 있으면 참    |  (a in list) = False   |
|  not in  | list내에 포함되어 있지 않으면 참 | (b not in list) = True |
